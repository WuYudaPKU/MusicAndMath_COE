在运行前，请保证`midiutil`包已下载。只需要`pip install midiutil`即可。

代码架构是这样的：所有有关训练和音乐的参数，在`config.py`中有很详细的注释介绍，可以通过调整`config.py`来调整训练偏好；

`util.py`中是调用的工具函数，不要修改；`fitness_function.py`中是适应度函数，可以进行修改。可以按照任意和弦走向进行（运行后的交互式页面）。`main.py`是一个乐句的训练主程序，启动`main.py`会开始训练，给出一段四个小节的乐句。`composer.py`是组成乐章的核心函数，可以先生成旋律A，然后生成变体A'，接着给出B，最后回到A。总体效果比一个乐句要好很多。

最佳样本，也是报告中呈现的音乐，在`.\Best_Results`中。也可以听一听nice_results中的音乐，或者直接运行`composer.py`看新生成的音乐效果如何，实测目前的参数已经达到了每次生成的乐曲都好听的水平。
